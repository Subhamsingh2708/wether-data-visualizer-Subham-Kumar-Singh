This project loads, cleans, analyzes, and visualizes weather data using Pandas, NumPy, and Matplotlib.
It automatically detects column names, handles missing fields, and generates useful insights.

Features

Automatic column detection (date, temperature, humidity, rainfall)

Cleans and standardizes dataset

Generates synthetic rainfall if missing

Computes daily, monthly, and seasonal statistics

Creates multiple visualizations:

Daily temperature trend

Monthly rainfall totals

Humidity vs temperature scatter plot

Combined temperature + rainfall graph
